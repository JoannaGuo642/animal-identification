# 多类别动物智能识别可视化系统
## 项目概述
本项目基于YOLOv8轻量化模型开发桌面端动物识别软件，针对人工观测统计效率低、易出错问题，实现图片批量处理、摄像头实时动物检测，搭配可视化统计面板与报表导出功能，低配电脑CPU即可流畅运行，适用于户外观测、动物素材分类场景。

## 技术栈
Python、YOLOv8n、OpenCV、PyQt5、QThread多线程、Excel/TXT文件读写

## 核心功能
1. 四层分层架构：界面交互层、业务逻辑层、检测模型层、硬件资源层，代码解耦易维护扩展；
2. 多模式输入识别：支持单张图片、批量文件夹、本地摄像头实时视频流三种识别方式；
3. 动态置信度调节：滑块自由调整识别阈值，平衡漏检与误检，适配不同拍摄环境；
4. 多线程任务处理：推理、摄像头采集放在子线程，避免GUI界面卡顿无响应；
5. 自动物种统计：实时统计各类动物数量，生成可视化统计看板，直观展示识别分布；
6. 完整数据归档：一键导出Excel/TXT识别日志，支持保存带检测框标注图片、留存历史记录；
7. 轻量化CPU适配：选用YOLOv8n轻量网络，优化内存占用，普通电脑低负载稳定运行。

## 仓库文件说明
- yolov8.py：模型训练、图像推理、报表导出底层核心工具类
- main.py：Qt主界面逻辑，绑定所有按钮、摄像头、批量识别交互事件
- Ui_Dialog.py：Qt Designer生成可视化界面布局文件
- *.pt：训练完成的自定义动物检测模型权重
- 动物识别检测.pptx：项目架构、训练实验对比、功能演示完整汇报文档
